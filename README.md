# `@erictaylor/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ yarn add --dev @erictaylor/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@erictaylor/prettier-config"
}
```

_alternatively_

```js
# .prettierrc.js
module.exports = {
  ...require('@erictaylor/prettier-config');
};
```
